-------- Contacts.com ® --------

A aplicação Contacts.com ®, cria, gerência e compartilha listas com contatos entre os mais diferentes setores de uma empresa / organização, agilizando a busca de contatos e consequentemente a troca de informações. 
Com essa aplicação problemas como listas telefonicas desatualizadas e gastos desnecessarios como o descarte de grandes volumes de papeis gerados após uma atualização de listas acabaram. 

Muitas empresas criam  suas listas de contatos em documentos do Microsoft Excel, imprimem e  as distribuem entre os diferentes setores. 

Quais os problemas gerados por isso?

-- Desperdicio / gasto desnecessário de folhas;
-- Rápida defasagem dessas listas;
-- Dificil atualização.

Sem contar que isso é um processo arcáico e ultrapassado, que apenas gera custos desnecessários e desencontro de informações.

Contacts.com ® tem uma interface amigavel e de facil compreenção, tudo para facilitar a vida do usuário.

Para uma gestão segura dos contatos, todos os usuarios do sistema são cadastrados e recebem um logim e senha de acesso, com isso apenas usuarios autorizados podem editar ou apagar contatos.

O cadastro de um novo contato deve conter apenas informações relevantes, que visem a agilidade do processo na hora de criar/editar um registro.
Pesquisas recentes revelam que usuários não avaliam positivamente cadastros longos e com informações irrelevantes.

Dados armazenados com  o sistema Contacts.com ®. 
- Nome
- Setor 
- Ramal
- Celular
- E-mail
 
Para a próxima versão estão sendo inseridas novas funcionalidades, uma delas é uma area reservada para criação de pequenos recados, essa por sua vez terá o formato de pequenas notas adesivas onde o usuário poderá salvar de forma rápida e prática recadinhos direto na tela do seu computador.
